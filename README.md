# scai_tutor_mobile

Front end de l'appli scai tutor. Développé par CoTIA & Partenaire
## Getting Started

Pour faire fonctionner ce projet, creez un fichier .env dans lequel vous copierez le contenu du fichier .env.example.

Ensuite, il suffit de fournir la bonne clé ```HF_TOKEN``` et le tour est joué.

Lancez alors le projet en connectant votre appareil mobile ou votre émulateur android.
